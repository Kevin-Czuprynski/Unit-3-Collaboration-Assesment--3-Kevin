### Tree Identification Tool
You can use this simple **Javascript** functon to identify a tree by its characteristics:

```
function identify_tree (leaf_shape, region) {
    if (leaf_shape == "needle" && region == "cold"){
        return "Pine Tree"
    } else if (leaf_shape == "bread" && region == "temperate"){
        return "Oak Tree"
    }else{
        Return "Unknown Tree"
    }
}
